## API

`GET /orders` - get all orlders

`GET /order/<orderID>` - get orlder by ID

`POST /orders/insert` - add new order

`POST /orders/delete` - remove order

`GET /pizza` - get all pizzas

`POST /pizza/insert` - add new pizza
